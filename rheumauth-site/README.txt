RheumAuth website files

This folder contains a very simple starter website.

What to upload to GitHub:
- index.html

How to launch it:
1. Open your empty GitHub repository
2. Click "uploading an existing file"
3. Drag index.html into GitHub
4. Commit the file
5. Go to Vercel
6. Add New -> Project
7. Import your RheumAuth GitHub repo
8. Deploy
9. In Vercel, add your domain RheumAuth.com
10. Follow the DNS instructions Vercel gives you
